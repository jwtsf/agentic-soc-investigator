# SOC Documentation Repository

This repository contains **Security Operations Center (SOC)** incident investigation reports created as part of hands-on analysis and learning exercises.

Each incident report follows a **structured, analyst-grade format** suitable for SOC documentation, interviews, and portfolio review.

---

##  Repository Structure

- **Incident_1** – Phishing Alert Investigation (Deceptive Email)
- **Incident_2** – Privilege Escalation via CVE Exploitation
- **Incident_3** – SharePoint ToolShell Authentication Bypass & RCE
- **Incident_4** – Lumma Stealer via ClickFix Phishing (DLL Side-Loading)
- **Incident_5** – Windows OLE Zero-Click Remote Code Execution
- **Incident_6** – Impersonating Domain MX Record Change Detection
- **Incident_7** – Palo Alto PAN-OS Command Injection (CVE-2024-3400)
- **Incident_8** – SQL Injection & XSS Attack on Web Server
- **Incident_9** – RDP Brute Force Attack Detection
- **Incident_10** – Malicious Macro Execution Leading to PowerShell Payload

---

##  Contents of Each Incident

Each incident report includes:

- Alert overview & metadata  
- Timeline of events  
- Evidence screenshots  
- Log and artifact analysis  
- Endpoint / network investigation  
- Indicators of Compromise (IOCs)  
- MITRE ATT&CK mapping  
- Response & remediation steps  
- Final analyst verdict  

---

##  Purpose

- Demonstrate **SOC analyst investigation workflow**
- Practice **incident triage, analysis, and documentation**
- Build a **professional SOC portfolio**
- Improve understanding of **real-world attack techniques**

>  All data is sanitized and simulated.  
> These reports are created **strictly for learning and portfolio purposes**.
